# Retail Store — Exploratory Data Analysis

End-to-end EDA on a retail sales dataset using Python, 
performed in Jupyter Notebook.

## What It Does
- Identifies sales trends across time periods
- Detects seasonal patterns in customer purchasing
- Finds outliers in sales data
- Produces visual summary report with charts

## Tech Stack
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Key Findings
- Identified peak sales seasons
- Detected underperforming product categories
- Visualized monthly revenue trends
